Editores Editores de Código

VSC

Atom

es un moderno editor de código y texto que además es hackeable y que ha sido creado por la gente de Github.

sublime

Sublime Text es un editor de Texto para escribir código en casi cualquier formato de archivo.

notepad ++

Notepad++ es un editor de texto y de código fuente libre con soporte para varios lenguajes de programación.
